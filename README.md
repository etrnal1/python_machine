### python 系列的计算

序列就是列表 序列里面的计算 就是列表的计算

|      |      |                   |
| ---- | ---- | ----------------- |
| 索引 | []   | 取序列中元素      |
| 连接 | +    | 将序列连接在一起  |
| 重复 | *    | 重复N次连接       |
| 成员 | in   | 列表是否包含元素  |
| 长度 | len  | 取出序列的长度    |
| 切片 | [:]  | 取出序列的 一部分 |
